# cross-memory 🧠

> 跨工作空间全局记忆检索 — 让 WorkBuddy 在任何工作空间都能记住你

## 这是什么

WorkBuddy 的记忆默认只在当前工作空间内有效。这个 Skill 打破了这个限制，让你在任何工作空间的对话中都能访问完整的个人记忆。

**核心功能：**
- 自动发现并读取全局记忆中心（MEMORY.md）
- 跨工作空间检索详细信息
- 即时写入重要新信息
- 与 Dream Skill 配合实现自动记忆蒸馏

## 架构

```
┌─────────────────────────────────────────────┐
│  Layer 1: 全局画像（平台自动注入）            │
│  ~/.workbuddy/memery/ → 所有空间基础上下文   │
└─────────────────────────────────────────────┘
                    ↓ 补充
┌─────────────────────────────────────────────┐
│  Layer 2: cross-memory Skill（按需深度检索）  │
│  ~/.workbuddy/memory/MEMORY.md → 完整记忆    │
│  ├─ 当前状态（工作/健康/财务/...）            │
│  ├─ 稳定认知（画像/风格/偏好/...）            │
│  ├─ 关系与背景                               │
│  └─ 跨工作空间记忆地图                        │
└─────────────────────────────────────────────┘
```

## 快速开始

### 1. 安装

将 `cross-memory` 文件夹复制到：
```
~/.workbuddy/skills/cross-memory/
```

### 2. 创建全局记忆中心

```bash
# 创建目录
mkdir -p ~/.workbuddy/memory

# 创建 MEMORY.md（按模板填写你的信息）
```

MEMORY.md 模板见 SKILL.md 末尾的"MEMORY.md 模板"章节。

### 3. 使用

在任何工作空间的对话中说：
- "你了解我什么？"
- "查一下我的记忆"
- "之前我说过什么关于投资的事？"
- "我的健康状况怎么配合工作安排？"

AI 会自动触发 cross-memory Skill，读取你的全局记忆并回答。

## 配合 Dream Skill（推荐）

安装 [Dream Skill](https://github.com/...) 后，可以实现：
- 每日自动蒸馏对话记忆
- 增量更新 MEMORY.md
- 过期信息归档到永久档案
- 全局画像自动同步

## 依赖

- WorkBuddy 平台
- 可选：Dream Skill（自动蒸馏）
