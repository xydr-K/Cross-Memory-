# 跨对话记忆整合 🧠

> 让 WorkBuddy 在任何工作空间都能记住你——跨工作空间全局记忆检索 Skill

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![WorkBuddy](https://img.shields.io/badge/WorkBuddy-Compatible-blue)]()

## 这是什么？

WorkBuddy 的记忆默认只在**当前工作空间**内有效。你在"投资"空间聊的内容，"健康"空间是看不到的。

**cross-memory** 打破了这个限制。它是一个 WorkBuddy Skill，通过建立全局记忆中心（`MEMORY.md`），让你在**任何工作空间**的对话中都能访问完整的个人上下文。

### 核心能力

- 🧠 **全局记忆读取** — 任何工作空间都能获取你的完整个人画像
- 🔍 **动态发现** — 自动扫描所有 `.workbuddy/memory/` 目录，无需硬编码路径
- ✍️ **即时写入** — 重要决策/纠正直接更新，不用等每日蒸馏
- 🔗 **跨域关联** — 自动关联跨领域信息（健康影响投资、工作影响时间安排）
- 🔄 **增量更新** — 日常信息追加到日报，Dream 蒸馏自动整合

### 适用场景

| 场景 | 没有 cross-memory | 有 cross-memory |
|------|-------------------|----------------|
| 投资空间问"我尿酸高能吃什么药？" | ❌ 不知道你的健康状况 | ✅ 读取健康档案给出建议 |
| 健康空间问"根据我的财务情况推荐体检方案？" | ❌ 不知道你的财务状况 | ✅ 结合预算推荐 |
| 新工作空间问"帮我写周报" | ❌ 不知道你的团队和格式 | ✅ 基于团队背景生成 |
| 任何空间说"我之前说过..." | ❌ 只能看当前空间历史 | ✅ 检索全局记忆 |


## 快速开始

### 1. 安装

将 `cross-memory/` 文件夹复制到你的 WorkBuddy Skills 目录：

```
~/.workbuddy/skills/cross-memory/
```

文件结构：
```
cross-memory/
├── SKILL.md              # Skill 定义（核心文件）
├── README.md             # 使用说明（本文件）
├── _meta.json            # Skill 元数据
└── _skillhub_meta.json   # Skill Hub 元数据
```

### 2. 初始化记忆中心

创建全局记忆目录和文件：

```bash
# 创建目录
mkdir -p ~/.workbuddy/memory

# 创建 MEMORY.md（复制下方模板内容）
```

在 `MEMORY.md` 中填写你的个人信息。模板如下：

```markdown
# 全局记忆中心

## 当前状态

**工作**
- （填写工作信息，如公司、团队、当前项目）

**健康**
- （填写健康状况，如慢性病、用药、体检指标）

**财务**
- （填写财务概况，如收入、资产分布、投资偏好）

## 稳定认知

**个人画像**
- （填写基本信息，如年龄、城市、性格）

**工作风格**
- （填写工作偏好，如沟通方式、交付标准）

**技术偏好**
- （填写技术栈和工具偏好）

**兴趣**
- （填写兴趣爱好）

## 关系与背景
- （填写团队、家族、公司等背景信息）

## 跨工作空间记忆地图

| 工作空间 | 路径 | 记忆焦点 |
|---------|------|---------|
| 全局记忆 | ~/.workbuddy/memory/ | 综合记忆中心 |
| （添加更多） | | |

## Dream
- （由 Dream Skill 自动维护蒸馏状态）
```

### 3. 使用

安装完成后，在任何工作空间中说以下任意一句话即可触发：

- "你了解我什么？"
- "查一下我的记忆"
- "之前我说过什么关于投资的事？"
- "根据我的健康状况，这个方案可行吗？"
- "我的团队背景是什么？"

AI 会自动加载 cross-memory Skill，读取你的全局记忆并给出个性化回答。

## 配合 Dream Skill（推荐进阶）

如果同时安装 [Dream Skill](https://github.com/)，可以实现**全自动记忆管理**：

| 功能 | 说明 |
|------|------|
| 每日蒸馏 | 自动将当天对话中的新信息提取到 MEMORY.md |
| 增量更新 | 只更新变化的内容，不重复写入 |
| 过期归档 | 已完成/过期的信息移入永久档案，不丢失 |
| 全局同步 | 自动同步精华摘要到平台全局画像 |

### 自动化蒸馏配置

Dream 可以配置为每日定时执行（如每天 18:00），实现无人值守的记忆管理。

## 文件说明

| 文件 | 用途 |
|------|------|
| `SKILL.md` | Skill 定义文件，包含触发场景、使用流程、模板 |
| `README.md` | 项目说明文档 |
| `_meta.json` | Skill 版本和元数据 |
| `_skillhub_meta.json` | Skill Hub 展示信息 |

## 工作原理

1. **触发** — 用户对话中出现记忆相关关键词时，AI 自动加载 Skill
2. **发现** — Skill 动态扫描用户系统中所有 `.workbuddy/memory/MEMORY.md`
3. **读取** — 加载全局记忆中心的完整内容
4. **深度检索** — 根据需要读取子工作空间的详细记忆
5. **写入** — 新信息即时写入或等待蒸馏整合
